Pythonで学ぶプログラミング

「Pythonで学ぶプログラミング」は、関が担当する東洋大学経営学部3年次配当科目「プログラミング実習講義」の授業テキストです。プログラミング未経験者がプログラミングの初歩を理解できるように、Pythonのプログラムを動かしながら学びます。

「プログラミング実習講義」の履修者は、授業の進め方についてガイダンスをよく読み、第1回の授業で配信されるガイダンス動画を視聴してください。ガイダンスの内容を理解していないことによる成績判定に関する申し立ては受け付けません。
